# EcoQA Benchmark Results

- `model_comparison.csv`: Aggregated summary for each evaluated model run.
- `*_details.jsonl`: Per-trajectory records generated by `run_ecoqa_benchmark.py`.

Each summary row corresponds to one model profile run over one dataset split.

Key columns:
- `pass_at_1`, `pass_at_k`, `reward_mean`
- `acc_structure` (primary accuracy under unified `{"items":[...]}` answers)
